# Arkansas Population Density by County (2020 Census)

This map visualizes population density by county in Arkansas using the USA 2020 Census Population Characteristics layer from ArcGIS Living Atlas.

## Purpose
This project demonstrates:
- Creating a choropleth map in ArcGIS Online
- Adding readable county labels and a state outline
- Cleaning popups for clarity
- Publishing a public-facing GIS map
